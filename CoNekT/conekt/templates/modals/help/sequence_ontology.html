Sequence Ontology (SO)

Sequence Ontology provides a standardized vocabulary to describe biological sequence features, particularly for transposable elements (TEs).

What is Sequence Ontology?

The Sequence Ontology (SO) is a structured controlled vocabulary for sequence features used in biological sequence annotation. It provides precise definitions for genomic and proteomic features.

How it works in this database:
  • SO Terms are associated with TE Classes
  • Each TE Class can have one or more SO term annotations
  • Sequences are linked to SO terms via their TE Class associations
Key Information:
  • SO ID: Unique identifier (e.g., SO:0000180)
  • Name: Standard term name (e.g., "retrotransposon")
  • Namespace: Category of the term
  • Synonyms: Alternative names and aliases
  • TE Classes: Associated transposable element classes
Example Applications:
  • Standardized annotation of repetitive elements
  • Classification of transposable elements
  • Cross-reference with other genome databases
  • Comparative genomics studies
